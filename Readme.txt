NextGen Gallery gallery Public Deleter on your homepage.

Usage:

**************Show Deleter using php code*********************

<?php show_ngg_gallery_pd($gid) gid refers to the gallery id

**************Show scroller using shortcode in content********
[ngd_show id=xxx] xxx refers to the gallery id


**************Usage*******************************************
A list of all images name within the gallery will be shown according to the gallery id selected.

check the image checkbox on the right and press delete. A simple message will show up upon sccuessful deleting the images.
